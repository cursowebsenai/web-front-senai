DOM Document Object Model

O DOM é uma das APIs mais usadas na web, permitindo que o javascript acesse e altere todos os elementosde um documento HTML

Principais usos do DOM

  • Criar páginas inteligentes
  • Criar aplicações inteligentes
  • Modificar o conteúdo ou o estilo de página sem precisar recarregá-la

Como DOM é criado?

  • Os desenvolvedores podem adicionar manipuladores de eventos aos elementos DOM
  • Para isso, é preciso usar métodos como addEventListener ou atributos de eventos diretamente no HTML
